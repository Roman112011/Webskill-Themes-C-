Змінні, Типи Даних та I/O (Input/Output)

Змінні та Типи Даних

Змінні, Типи Даних та як з ними працювати.

Змінні:

Змінні - це контейнери, що використовуються для зберігання та управління даними в програмі. У C# імена змінних повинні бути унікальними і можуть містити літери, цифри і символ підкреслення, починатися вони повинні з літери.

int age;  // Приклад оголошення змінної типу int (ціле число)
string name;  // Приклад оголошення змінної типу string (рядок)

Типи Даних:

Тип даних визначає, які види значень можуть бути збережені в змінних. Основні типи даних в C# включають:

  1. int - для зберігання цілих чисел. Наприклад, int age = 10;.

  2. float - для зберігання чисел з плаваючою точкою (дійсних чисел). Наприклад, float temperature = 98.6f;.

  3. double - для точних чисел з плаваючою точкою. Наприклад, double pi = 3.14159265359;.

  4. bool - для логічних значень True або False. Наприклад, bool isRaining = true;.

  5. char - для одного символу. Наприклад, char grade = 'A';.

  6. string - для рядків тексту. Наприклад, string message = "Привіт, світе!";.

  7. decimal - для фінансових обчислень, де потрібна велика точність. Наприклад, decimal balance = 1000.50m;.

  8. DateTime - для роботи з датами та часом. Наприклад, DateTime currentDate = DateTime.Now;.

Змінні можуть бути ініціалізовані, тобто присвоєні початкове значення під час їх оголошення. Для ініціалізації змінної використовується оператор присвоєння (=). Наприклад:

int x = 5;           // Змінна x отримала значення 5.
string name = "John"; // Змінна name отримала значення "John".

Після ініціалізації значення змінної може бути змінено, наприклад:

x = 10;           // Змінна x тепер має значення 10.
name = "Alice";   // Змінна name тепер має значення "Alice".

Введення/Виведення (I/O)

У мові програмування C# введення та виведення інформації використовується для взаємодії з користувачем або іншими програмами.

Виведення Даних (Output):

Для виведення даних на екран використовується об'єкт типу Console. Основний метод для виведення - це WriteLine, який виводить рядок тексту на екран і переводить курсор на новий рядок:

Console.WriteLine("Привіт, світе!"); // Виведе "Привіт, світе!" на екрані.

Також можна використовувати Write, який не переводить курсор на новий рядок, із чимось іншим:

Console.Write("Це "); 
Console.Write("приклад."); 
// Виведе "Це приклад." на одному рядку.

Введення Даних (Input):

Для отримання даних від користувача можна використовувати методи класу Console.ReadLine(). Цей метод зчитує рядок тексту, введений користувачем, і повертає його як рядок. Цей рядок можна конвертувати в відповідний тип даних, якщо потрібно.

Console.Write("Введіть ваше ім'я: ");
string name = Console.ReadLine();
Console.WriteLine($"Привіт, {name}!");

Цей код спробує зчитати рядок тексту, який користувач введе після запиту імені, і виведе привітання з цим іменем.

Форматування Виведення:

Мова C# надає різні способи форматування виведення даних. Один зі способів - використання рядкових специфікаторів. Наприклад:

int age = 25;
Console.WriteLine("Мені {0} років.", age);

У цьому прикладі {0} - це заповнювач, який буде замінений значенням age.

Це загальний огляд введення та виведення даних в C#. В майбутніх уроках ви дізнаєтеся більше про цю тему, включаючи роботу з файлами та роботу з рядками (strings).

Конвертація Типів Даних

У C#, інколи може знадобитися конвертувати один тип даних в інший. Це корисно, коли ви маєте дані одного типу і хочете використати їх як інший тип.

Різниця між явною та неявною конвертацією полягає у рівні контролю над процесом конвертації та можливостях обробки помилок. Ось докладне пояснення, чому і коли використовувати кожен вид конвертації, з прикладами:

Явна (Explicit) Конвертація (кастинг):

  • Контроль: Явна конвертація дозволяє розробнику вказати точно, який тип конвертації потрібно виконати. Це дає більший контроль над процесом конвертації.

  • Можливість втрати даних: Якщо конвертація приводить до втрати даних (наприклад, змінюємо double на int), вона буде виконана, і дані будуть обрізані до типу призначення без попереджень.

Приклад явної конвертації:

double doubleValue = 10.5;
int intValue = (int)doubleValue; // Явна конвертація до int.

Неявна (Implicit) Конвертація:

  • Контроль: Неявна конвертація виконується автоматично, і компілятор сам визначає, чи може вона бути виконана без втрати даних.

  • Можливість втрати даних: Якщо конвертація приводить до втрати даних, вона може призвести до виключення (exception).

Приклад неявної конвертації:

int intValue = 5;
double doubleValue = intValue; // Неявна конвертація до double.

Коли використовувати кожен вид конвертації:

  • Використовуйте явну конвертацію, коли ви впевнені, що конвертація не призведе до втрати даних, і ви хочете точно визначити тип конвертації.

  • Використовуйте неявну конвертацію, коли компілятор може автоматично визначити правильний тип конвертації та ви впевнені, що втрати даних не виникнуть.

  • Якщо є можливість втрати даних і важливо перевірити результат, використовуйте методи конвертації або Parse методи, оскільки вони можуть повернути помилку, якщо конвертація неможлива або призводить до втрати даних.

Пам'ятайте, що при невірній конвертації можуть виникати помилки в час виконання програми, тому оберіть метод конвертації залежно від конкретної ситуації та потреб вашої програми.

Оператори в C# та їх використання

Оператори - це символи чи ключові слова, які використовуються для виконання різних операцій у програмах. В C# існує безліч операторів для виконання арифметичних, логічних та порівняльних операцій. Ось докладний огляд найпоширеніших операторів разом з прикладами їх використання.

  1. Арифметичні Оператори:

    • + (додавання): Додає два числа разом.

      int result = 5 + 3; // result = 8

    • - (віднімання): Віднімає одне число від іншого.

      int result = 8 - 3; // result = 5

    • * (множення): Перемножує два числа.

      int result = 4 * 3; // result = 12

    • / (ділення): Ділить одне число на інше.

      int result = 10 / 2; // result = 5

  2. Логічні Оператори:

    • && (логічне І): Повертає true, якщо обидва операнди true.

      bool result = true && false; // result = false

    • || (логічне АБО): Повертає true, якщо хоча б один операнд true.

      bool result = true || false; // result = true

    • ! (логічне НЕ): Повертає обернене значення операнду.

      bool result = !true; // result = false

  3. Порівняльні Оператори:

    • > (більше): Перевіряє, чи одне число більше за інше.

      bool result = 5 > 3; // result = true

    • < (менше): Перевіряє, чи одне число менше за інше.

      bool result = 5 < 3; // result = false

    • >= (більше або дорівнює): Перевіряє, чи одне число більше або дорівнює іншому.

      bool result = 5 >= 5; // result = true

    • <= (менше або дорівнює): Перевіряє, чи одне число менше або дорівнює іншому.

      bool result = 3 <= 5; // result = true

    • == (рівно): Перевіряє, чи два значення рівні.

      bool result = 5 == 5; // result = true

    • != (не дорівнює): Перевіряє, чи два значення не рівні.

      bool result = 5 != 3; // result = true

  4. Присвоєння Операторів:

    • = (присвоєння): Присвоює значення правого операнда змінній лівого операнда.

      int x = 10;

    • += (додавання та присвоєння): Додає значення правого операнда до змінної лівого операнда і присвоює результат змінній лівого операнда.

      int x = 5;
x += 3; // x = 8

    • -= (віднімання та присвоєння): Віднімає значення правого операнда від змінної лівого операнда і присвоює результат змінній лівого операнда.

      int x = 10;
x -= 4; // x = 6

  5. Інші Оператори:

    • % (ділення залишком): Повертає залишок від ділення лівого операнда на правий.

      int result = 10 % 3; // result = 1

    • ?: (тернарний оператор): Використовується для умовної операції.

      int x = 5;
int y = (x > 3) ? 10 : 20; // y = 10, якщо x > 3, інакше y = 20

Це основні оператори в мові C#, які використовуються для різних видів операцій в програмах. Вони важливі для розробки алгоритмів та обчислень в вашому коді.
