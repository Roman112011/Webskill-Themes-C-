Масиви (Arrays) - Двовимірні

Масиви є структурами даних, які дозволяють зберігати більше одного значення в одній змінній. Двовимірні масиви, відомі також як матриці, дозволяють зберігати дані в двовимірній таблиці.

// Оголошення двовимірного масиву (матриці) 3x3
int[,] matrix = new int[3, 3];

// Ініціалізація значень в матриці
matrix[0, 0] = 1;
matrix[0, 1] = 2;
matrix[0, 2] = 3;
matrix[1, 0] = 4;
matrix[1, 1] = 5;
matrix[1, 2] = 6;
matrix[2, 0] = 7;
matrix[2, 1] = 8;
matrix[2, 2] = 9;

// Доступ до значень матриці
int element = matrix[1, 1]; // element буде 5

Колекції (Collections) - Двовимірні (List)

Колекції - це розширені структури даних, які дозволяють зберігати та управляти динамічною кількістю елементів. У C#, ви можете використовувати List для створення двовимірних колекцій.

// Оголошення двовимірного List
List<List<int>> matrixList = new List<List<int>>();

// Додавання списків до двовимірного List
matrixList.Add(new List<int> { 1, 2, 3 });
matrixList.Add(new List<int> { 4, 5, 6 });
matrixList.Add(new List<int> { 7, 8, 9 });

// Доступ до значень колекції
int element = matrixList[1][1]; // element буде 5

Різниця між Масивами та Колекціями

  • Розмір: Масиви мають фіксований розмір, коли вони створюються, і не можуть змінювати його. Колекції, такі як List, динамічні і можуть змінюватися в розмірі за потребою.

  • Синтаксис: Робота з масивами є більш статичною, оскільки їх розмір визначається під час оголошення. Колекції, з іншого боку, дозволяють додавати та видаляти елементи без зміни розміру колекції.

  • Використання: Масиви зазвичай використовуються, коли потрібен статичний набір даних з фіксованим розміром. Колекції надають більшу гнучкість для зберігання та роботи з динамічними даними.

Двовимірні Масиви

Використання двовимірних масивів.

Двовимірні масиви, також відомі як матриці, дозволяють зберігати дані в двовимірній структурі. Ось як використовувати двовимірні масиви в C# з поясненнями:

Оголошення та створення двовимірного масиву

// Оголошення двовимірного масиву цілих чисел 3x3
int[,] matrix = new int[3, 3];

Цей код оголошує двовимірний масив з іменем matrix розміром 3x3. Це означає, що в ньому можна зберігати 3 рядки та 3 стовпці.

Ініціалізація значень в масиву

matrix[0, 0] = 1;
matrix[0, 1] = 2;
matrix[0, 2] = 3;
matrix[1, 0] = 4;
matrix[1, 1] = 5;
matrix[1, 2] = 6;
matrix[2, 0] = 7;
matrix[2, 1] = 8;
matrix[2, 2] = 9;

Ці строки коду заповнюють двовимірний масив matrix числами від 1 до 9. Індекси [рядок, стовпець] використовуються для вказівки конкретної позиції в масиві.

Доступ до значень масиву

int element = matrix[1, 1];

Ця строка коду дозволяє отримати значення, яке знаходиться в другому рядку та другому стовпці масиву matrix. У цьому випадку значення буде 5.

Виведення значень масиву на екран

for (int row = 0; row < 3; row++)
{
    for (int col = 0; col < 3; col++)
    {
        Console.Write(matrix[row, col] + " ");
    }
    Console.WriteLine();
}

Ця пара вкладених циклів for дозволяє вивести всі значення з двовимірного масиву matrix на екран у вигляді матриці. Console.Write виводить значення зі змінної, а Console.WriteLine перехід на новий рядок після кожного рядка матриці.

Двовимірні Спики (List)

Використання двовимірних списків.

З двовимірними списками (List) у C# можна створювати та працювати зі структурами даних, подібними до двовимірних масивів, але з більшою гнучкістю.

Створення та Оголошення Двовимірного List:

// Оголошення та створення двовимірного List
List<List<int>> matrixList = new List<List<int>>();

// Додавання списків до двовимірного List
matrixList.Add(new List<int> { 1, 2, 3 });
matrixList.Add(new List<int> { 4, 5, 6 });
matrixList.Add(new List<int> { 7, 8, 9 });

Доступ до Значень та Редагування Двовимірного List:

// Доступ до значень у двовимірному List
int element = matrixList[1][1]; // element буде 5

// Зміна значення у двовимірному List
matrixList[0][2] = 10; // Змінимо 3 на 10 у першому списку першого рядка

Виведення Двовимірного List:

// Виведення значень двовимірного List
for (int i = 0; i < matrixList.Count; i++)
{
    for (int j = 0; j < matrixList[i].Count; j++)
    {
        Console.Write(matrixList[i][j] + " ");
    }
    Console.WriteLine();
}

Додавання та Видалення Рядків та Стовпців:

// Додавання нового рядка
matrixList.Add(new List<int> { 10, 11, 12 });

// Видалення рядка
matrixList.RemoveAt(1); // Видаляємо другий рядок

// Додавання нового стовпця
for (int i = 0; i < matrixList.Count; i++)
{
    matrixList[i].Add(13); // Додаємо новий елемент (стовпець) зі значенням 13
}

// Видалення стовпця
for (int i = 0; i < matrixList.Count; i++)
{
    matrixList[i].RemoveAt(1); // Видаляємо другий стовпець
}

Важливо пам'ятати, що двовимірні списки (List) дозволяють легко додавати та видаляти рядки та стовпці, що надає більшу гнучкість для роботи зі структурами даних.
