Файлова система/System.IO

Файлова система/System.IO - Вступ

C# має великий набір інструментів для роботи з файловою системою. Ці інструменти доступні завдяки простору імен System.IO, який надає можливості для створення, читання, запису та видалення файлів і папок.

Основні інструменти, які надає System.IO, включають:

  1. File: Цей клас дозволяє вам створювати, читати, записувати та видаляти файли. Він також дозволяє перевіряти наявність файлів та копіювати їх.

  2. Directory: Цей клас надає можливості для роботи з папками. Ви можете створювати, видаляти, переміщати та переглядати папки.

  3. Path: Клас Path надає допоміжні функції для роботи зі шляхами до файлів і папок. Ви можете використовувати його для об'єднання шляхів, отримання імені файлу або папки, та ін.

Ось приклади використання цих інструментів:

using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Створення нового файлу
        File.WriteAllText("example.txt", "Це зразок тексту.");

        // Перевірка наявності файлу
        if (File.Exists("example.txt"))
        {
            // Читання файлу
            string text = File.ReadAllText("example.txt");
            Console.WriteLine("Вміст файлу: " + text);
        }

        // Створення нової папки
        Directory.CreateDirectory("НоваПапка");

        // Перегляд вмісту папки
        string[] files = Directory.GetFiles("НоваПапка");
        string[] directories = Directory.GetDirectories("НоваПапка");

        Console.WriteLine("Файли в папці:");
        foreach (string file in files)
        {
            Console.WriteLine(file);
        }

        Console.WriteLine("Папки в папці:");
        foreach (string directory in directories)
        {
            Console.WriteLine(directory);
        }
    }
}

У цьому прикладі ми створюємо файл, перевіряємо наявність файлу, читаємо його вміст, створюємо папку та переглядаємо файли та папки в цій папці. System.IO надає різноманітні можливості для роботи з файловою системою в C#.

System.IO - File

Клас "File" в просторі імен "System.IO".

Клас "File" в просторі імен "System.IO" в C# надає інструменти для роботи з файлами. Він дозволяє створювати, читати, записувати, видаляти та взаємодіяти з файлами. Ось детальні пояснення та приклади використання цього класу:

  1. Створення файлу:Для створення нового файлу використовуйте метод File.Create(). Цей метод створює порожній файл або очищає існуючий файл. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Створення нового файлу або очищення існуючого
        File.Create("новий_файл.txt").Close();
    }
}

  2. Запис у файл:Щоб записати дані в файл, використовуйте метод File.WriteAllText(). Він записує текстові дані в файл. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Запис тексту в файл
        File.WriteAllText("зразок.txt", "Це приклад тексту для файлу.");
    }
}

  3. Читання файлу:Для читання вмісту файлу використовуйте метод File.ReadAllText(). Він повертає вміст файлу як рядок. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Читання вмісту файлу
        string text = File.ReadAllText("зразок.txt");
        Console.WriteLine("Вміст файлу: " + text);
    }
}

  4. Перевірка наявності файлу:Для перевірки наявності файлу використовуйте метод File.Exists(). Він повертає true, якщо файл існує, і false, якщо файл відсутній. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Перевірка наявності файлу
        if (File.Exists("зразок.txt"))
        {
            Console.WriteLine("Файл існує.");
        }
        else
        {
            Console.WriteLine("Файл не існує.");
        }
    }
}

  5. Видалення файлу:Для видалення файлу використовуйте метод File.Delete(). Він видаляє файл з файлової системи. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Видалення файлу
        File.Delete("зразок.txt");
    }
}

Клас "File" надає ряд корисних методів для роботи з файлами в C# та дозволяє легко взаємодіяти з файловою системою.

System.IO - Directory

Клас "Directory" в просторі імен "System.IO".

Клас "Directory" в просторі імен "System.IO" в C# надає інструменти для роботи з папками (каталогами). Він дозволяє створювати, видаляти, переміщати, переглядати та взаємодіяти з папками в файловій системі. Ось детальні пояснення та приклади використання цього класу:

  1. Створення папки:Для створення нової папки використовуйте метод Directory.CreateDirectory(). Цей метод створює нову папку з вказаним ім'ям. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Створення нової папки
        Directory.CreateDirectory("Нова_папка");
    }
}

  2. Видалення папки:Для видалення папки використовуйте метод Directory.Delete(). Видалення папки може бути небезпечним, тому будьте обережні. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Видалення папки (якщо папка порожня)
        Directory.Delete("Нова_папка");
    }
}

  3. Перегляд вмісту папки:Ви можете переглядати файли та папки, які містяться в папці, за допомогою методів Directory.GetFiles() та Directory.GetDirectories(). Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Перегляд вмісту папки
        string[] files = Directory.GetFiles("Папка_з_файлами");
        string[] directories = Directory.GetDirectories("Папка_з_папками");

        Console.WriteLine("Файли в папці:");
        foreach (string file in files)
        {
            Console.WriteLine(file);
        }

        Console.WriteLine("Папки в папці:");
        foreach (string directory in directories)
        {
            Console.WriteLine(directory);
        }
    }
}

  4. Перевірка наявності папки:Для перевірки наявності папки використовуйте метод Directory.Exists(). Він повертає true, якщо папка існує, і false, якщо папка відсутня. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Перевірка наявності папки
        if (Directory.Exists("Папка_з_файлами"))
        {
            Console.WriteLine("Папка існує.");
        }
        else
        {
            Console.WriteLine("Папка не існує.");
        }
    }
}

Клас "Directory" дозволяє вам легко взаємодіяти з папками в файловій системі та використовувати їх для організації файлів та інших даних на вашому комп'ютері.

System.IO - Path

Клас "Path" в просторі імен "System.IO".

Клас "Path" в просторі імен "System.IO" в C# надає інструменти для роботи зі шляхами до файлів і папок. Він дозволяє вам створювати, об'єднувати, отримувати інформацію про шляхи та виконувати інші операції пов'язані з роботою із шляхами. Ось детальні пояснення та приклади використання класу "Path":

  1. Об'єднання шляхів:Використовуючи метод Path.Combine(), ви можете об'єднати декілька шляхів в один. Це корисно, коли вам потрібно створити шлях до файлу або папки, об'єднавши декілька частин шляху. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        string folder = "Моя_Папка";
        string filename = "файл.txt";

        string fullPath = Path.Combine(folder, filename);
        Console.WriteLine("Повний шлях: " + fullPath);
    }
}

  2. Отримання імені файлу або папки:Метод Path.GetFileName() дозволяє отримати ім'я файлу або папки з вказаного шляху. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        string filePath = "Папка_з_файлами/файл.txt";
        string fileName = Path.GetFileName(filePath);
        Console.WriteLine("Ім'я файлу: " + fileName);
    }
}

  3. Отримання імені розширення файлу:Метод Path.GetExtension() дозволяє отримати розширення файлу з вказаного шляху. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        string filePath = "Папка_з_файлами/файл.txt";
        string fileExtension = Path.GetExtension(filePath);
        Console.WriteLine("Розширення файлу: " + fileExtension);
    }
}

  4. Перевірка наявності неправильних символів у шляху:Метод Path.GetInvalidFileNameChars() дозволяє отримати масив символів, які не можуть бути використані у шляху до файлу або папки через обмеження операційної системи. Наприклад:

    using System;
using System.IO;

class Program
{
    static void Main()
    {
        char[] invalidChars = Path.GetInvalidFileNameChars();
        Console.WriteLine("Неправильні символи для імен файлів або папок:");
        foreach (char invalidChar in invalidChars)
        {
            Console.Write(invalidChar + " ");
        }
    }
}

Клас "Path" допомагає працювати зі шляхами до файлів і папок, забезпечуючи потрібну функціональність для обробки шляхів та роботи з файлами та папками відповідно до вимог вашої програми.

Приклад програми, яка використовує класи "File", "Directory" і "Path"

Приклад програми, яка використовує класи "File", "Directory" і "Path" для створення нової папки, запису текстового файлу в цій папці та відображення списку файлів у папці.

Приклад програми, яка використовує класи "File", "Directory" і "Path" для створення нової папки, запису текстового файлу в цій папці та відображення списку файлів у папці. Ця програма також використовує "Path" для об'єднання шляхів та отримання імені файлу:

using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Шлях до папки, яку ми створимо
        string folderPath = "Нова_Папка";

        // Створення нової папки
        Directory.CreateDirectory(folderPath);
        Console.WriteLine("Створено нову папку: " + folderPath);

        // Шлях до файла в новій папці
        string filePath = Path.Combine(folderPath, "новий_файл.txt");

        // Запис тексту в файл
        File.WriteAllText(filePath, "Це приклад тексту для файлу в новій папці.");
        Console.WriteLine("Записано текст у файл: " + filePath);

        // Перевірка наявності файла
        if (File.Exists(filePath))
        {
            Console.WriteLine("Файл існує.");
        }

        // Відображення імен файлів у папці
        string[] files = Directory.GetFiles(folderPath);
        Console.WriteLine("Файли в папці:");
        foreach (string file in files)
        {
            string fileName = Path.GetFileName(file);
            Console.WriteLine(fileName);
        }
    }
}

Ця програма виконує наступні дії:

  1. Створює нову папку з іменем "Нова_Папка".
  2. Об'єднує шляхи до папки та файлу в новій папці.
  3. Записує текстовий файл з текстом-прикладом в папку.
  4. Перевіряє наявність створеного файла в папці та виводить відповідне повідомлення.
  5. Виводить імена всіх файлів у створеній папці.

Ця програма демонструє, як використовувати класи "File", "Directory" та "Path" для взаємодії з файловою системою, створення та роботи з файлами та папками, а також для опрацювання шляхів до них.
