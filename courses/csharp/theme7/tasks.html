Завдання 1.X

Прості завдання.

1.1. Створіть клас "Колір" (Color), який має поля для зберігання червоного (Red), зеленого (Green) та синього (Blue) компонентів кольору. Додайте метод для виведення інформації про кольори.

1.2. Створіть клас "Книга" (Book), який має поля для назви (Title), автора (Author) та року видання (Year). Створіть об'єкт класу та виведіть інформацію про книгу на консоль.

1.3. Розширте клас "Користувач" (User) з попереднього прикладу. Додайте до нього метод, який дозволяє змінювати ім'я користувача та виводити оновлену інформацію.

1.4. Створіть клас "Калькулятор" (Calculator), який має методи для додавання, віднімання, множення та ділення двох чисел. Введіть два числа з консолі та виконайте операції з допомогою цього класу.

1.5. Створіть клас "Коло" (Circle) з полем для радіуса. Додайте метод для обчислення площі кола (π * радіус^2). Створіть об'єкт класу та виведіть площу на екран.

Завдання 2.X

Середні завдання.

2.1. Створіть клас "Прямокутник" (Rectangle) з полями довжини та ширини. Додайте метод для обчислення площі та периметру прямокутника. Створіть об'єкт класу та виведіть обидва результати на консоль.

2.2. Розширте клас "Користувач" (User) з попереднього прикладу. Додайте до нього метод для зміни віку користувача та виведіть оновлену інформацію.

2.3. Створіть клас "Студент" (Student) з полями для імені (Name), оцінки (Grade) та статусу (Active). Додайте метод для виведення інформації про студента. Створіть об'єкт класу та виведіть інформацію.

2.4. Створіть клас "Квадрат" (Square) з полем для довжини сторони. Додайте метод для обчислення площі квадрата. Створіть об'єкт класу та виведіть площу на екран.

Завдання 3.X

Складні завдання.

3.1. Створіть клас "Комп'ютер" (Computer) з полями для назви (Name), типу процесора (CPUType) та об'єму оперативної пам'яті (RAM). Додайте метод для виведення інформації про комп'ютер. Створіть об'єкт класу та виведіть інформацію.

3.2. Розширте клас "Прямокутник" (Rectangle) з попереднього прикладу. Додайте до нього метод для зміни розмірів прямокутника та обчислення площі та периметру після змін.

3.3. Створіть клас "Банківський рахунок" (BankAccount) з полями для номеру рахунку (AccountNumber) та балансу (Balance). Додайте методи для зняття коштів та внесення коштів на рахунок. Перевірте, чи не виходить баланс за межі допустимого.
