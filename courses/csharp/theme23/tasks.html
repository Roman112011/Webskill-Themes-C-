Завдання

Завдання 1.X

Прості завдання.

1.1. Створіть програму, яка містить Grid із двома рядками та двома стовпцями. Розмістіть у кожній клітинці сітки TextBlock із текстом "Привіт, світ!".

1.2. Розширте попередню програму, щоб кожен TextBlock мав власний колір фону та тексту.

1.3. Створіть програму, яка містить Grid з одним рядком та трьома стовпцями. Розмістіть у кожній клітинці сітки кнопку (Button) із текстом "Клікніть мене".

1.4. Розширте попередню програму, щоб кожна кнопка мала власний колір фону та тексту.

1.5. Створіть програму, яка містить Grid із чотирма рядками та одним стовпцем. Розмістіть у кожній клітинці сітки TextBlock із номером рядка (1, 2, 3, 4).

Завдання 2.X

Середні завдання.

2.1. Створіть програму, яка містить Grid із трьома рядками та трьома стовпцями. Розмістіть у кожній клітинці сітки TextBlock із номером рядка та стовпця (наприклад, "Рядок: 1, Стовпець: 1").

2.2. Створіть програму, яка містить Grid із п'ятьма рядками та п'ятьма стовпцями. Розмістіть у кожній клітинці сітки кнопку із текстом "Клікніть мене".

Завдання 3.X

Складні завдання.

3.1. Створіть калькулятор.

Для підрахунку використовуйте наступну функцію:

static double Evaluate(string expression)
    {

        loDataTable.Rows.Add(0);
        return (double)(loDataTable.Rows[0]["Eval"]);
    }
