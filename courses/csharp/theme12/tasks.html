Завдання

Завдання 1.X

Прості завдання.

1.1. Створіть клас Book з властивостями Title і Author. Серіалізуйте об'єкт типу Book в JSON та збережіть його у файл "book.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про книгу на консоль.

1.2. Створіть декілька об'єктів типу Person, де кожен об'єкт має властивості Name та Age. Серіалізуйте ці об'єкти в масив JSON і збережіть у файл "people.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про всіх осіб на консоль.

1.3. Створіть клас Product з властивостями Name, Price та Quantity. Серіалізуйте список об'єктів Product в JSON та збережіть у файл "products.json". Потім зчитайте цей JSON з файлу та виведіть загальну вартість всіх продуктів.

1.4. Створіть клас Student з властивістю Name та Grades, яка є масивом оцінок. Серіалізуйте об'єкт Student в JSON та збережіть у файл "student.json". Потім зчитайте цей JSON з файлу та виведіть середній бал студента на консоль.

1.5. Створіть клас Car з властивостями Make, Model та Year. Серіалізуйте об'єкт Car в JSON та збережіть у файл "car.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про автомобіль на консоль.

Завдання 2.X

Середні завдання.

2.1. Створіть клас Store зі списком продуктів (клас з попереднього завдання). Серіалізуйте об'єкт Store в JSON та збережіть його у файл "store.json". Потім зчитайте цей JSON з файлу та виведіть список продуктів та їх загальну вартість.

2.2. Створіть клас Order з властивістю CustomerName та списком продуктів. Створіть декілька об'єктів Order, серіалізуйте їх у JSON та збережіть у файл "orders.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про кожне замовлення.

2.3. Створіть клас BlogPost з властивостями Title, Author та Content. Серіалізуйте об'єкт BlogPost в JSON та збережіть у файл "blogpost.json". Потім зчитайте цей JSON з файлу та виведіть всю інформацію про пост на консоль.

2.4. Створіть клас Movie з властивостями Title, Director та ReleaseYear. Серіалізуйте об'єкт Movie в JSON та збережіть у файл "movie.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про фільм.

2.5. Створіть клас Employee зі списком проєктів, над якими працює співробітник. Серіалізуйте об'єкт Employee в JSON та збережіть його у файл "employee.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про проєкти співробітника.

Завдання 3.X

Складні завдання.

3.1. Створіть клас Library зі списком книг (клас з першого завдання). Серіалізуйте об'єкт Library в JSON та збережіть його у файл "library.json". Потім зчитайте цей JSON з файлу та виведіть список всіх книг у бібліотеці.

3.2. Створіть клас School зі списком студентів (клас з першого завдання). Серіалізуйте об'єкт School в JSON та збережіть його у файл "school.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про всіх студентів.

3.3. Створіть клас Company зі списком працівників (клас з попереднього завдання). Серіалізуйте об'єкт Company в JSON та збережіть його у файл "company.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про всіх працівників та проєкти, над якими вони працюють.

3.4. Створіть клас Country зі списком міст та їх населенням. Серіалізуйте об'єкт Country в JSON та збережіть його у файл "country.json". Потім зчитайте цей JSON з файлу та виведіть інформацію про країну, її столицю та міста з населенням.

3.5. Створіть клас MusicAlbum зі списком пісень. Серіалізуйте об'єкт MusicAlbum в JSON та збережіть його у файл "album.json". Потім зчитайте цей JSON з файлу та виведіть список пісень у музичному альбомі.
