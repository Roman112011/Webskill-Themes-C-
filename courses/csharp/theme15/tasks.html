Завдання 1.X

Прості завдання.

Завдання 1: Розширення взаємодії гравця з світом.

1.1 Додавання діагонального руху

Опис завдання:Реалізувати можливість рухатись гравцю по діагоналі.

Кроки для вирішення:

  • Впровадження обробки діагонального вводу в UserInputManager.
  • Зміна логіки керування гравцем у методі GamePlayerInLoop.

1.2 Додавання смертельних об'єктів

Опис завдання:Створити об'єкти, які при контакті з гравцем призводять до завершення гри.

Кроки для вирішення:

  • Створення нового класу об'єкта, який визначає смертельні об'єкти та їхню взаємодію з гравцем.
  • Оновлення методу GamePointsInLoop для обробки взаємодії гравця з смертельними об'єктами.

Завдання 2.X

Середні завдання.

Завдання 2: Створення основи 2D платформера

2.1 Реалізація основ фізики платформера

Опис завдання:Створення фізичної моделі для гравця: гравітація, рух вліво-вправо та можливість стрибати.

Кроки для вирішення:

  • Додавання гравітації та обробки стрибків у методі GamePlayerInLoop.
  • Реалізація керування гравцем для руху вліво-вправо та стрибків через UserInputManager.

2.2 Створення блоків, на яких може стояти гравець

Опис завдання:Створення блоків, на яких може стояти гравець, формуючи основу для платформера.

Кроки для вирішення:

  • Розробка класу Block - статичний об'єкт, на якому може стояти гравець.
  • Оновлення логіки взаємодії гравця з блоками у методі GamePlayerInLoop.
