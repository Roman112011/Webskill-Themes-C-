Завдання 1.X

Прості завдання.

1.1. (Array) Завдання для масиву:

  • Створіть масив цілих чисел з трьома елементами.
  • Заповніть масив числами 10, 20 і 30.
  • Виведіть всі елементи масиву на екран.

1.2. (List) Завдання для списку:

  • Створіть список рядків і додайте до нього три різні назви фруктів.
  • Видаліть другий елемент зі списку.
  • Виведіть всі залишаються елементи на екран.

1.3. (Queue) Завдання для черги:

  • Створіть чергу, яка буде включати трьох різних студентів.
  • Додайте ще одного студента до черги.
  • Видаліть та виведіть першого студента з черги.

Завдання 2.X

Середні завдання.

2.1. (Array) Завдання для масиву:

  • Створіть масив дійсних чисел.
  • Заповніть масив довільними числами.
  • Знайдіть і виведіть середнє значення всіх чисел в масиві.

2.2. (List) Завдання для списку:

  • Створіть список цілих чисел і заповніть його послідовними натуральними числами від 1 до 10.
  • Видаліть всі парні числа зі списку.
  • Виведіть залишаються числа.

2.3. (Queue) Завдання для черги:

  • Створіть чергу для замовлень в ресторані.
  • Додайте три замовлення: "Піца", "Салат" і "Паста".
  • Перегляньте, яке замовлення буде обслуговуватися першим і виведіть його.
