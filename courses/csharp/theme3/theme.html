Умовні оператори

Огляд умовних операторів if, else if та else.

Умовний оператор if:

Оператор if використовується для виконання певної дії, якщо певна умова виконується. Синтаксис оператора if виглядає так:

if (умова)
{
    // Код, який виконається, якщо умова правдива
}

Умова - це вираз, який повертає булеве значення true або false. Якщо умова правдива, то код, розміщений у фігурних дужках, виконується. В іншому випадку, код ігнорується.

Приклад:

int age = 12;

if (age < 18)
{
    Console.WriteLine("Ви ще не досягли повноліття.");
}

Умовний оператор else if:

Оператор else if дозволяє перевіряти додаткові умови, якщо перша умова в операторі if не виконується. Ви можете мати багато операторів else if після початкового if. Синтаксис виглядає так:

if (умова1)
{
    // Код, який виконається, якщо умова1 правдива
}
else if (умова2)
{
    // Код, який виконається, якщо умова2 правдива
}
else
{
    // Код, який виконається, якщо жодна з умов не виконується
}

Приклад:

int age = 16;

if (age < 12)
{
    Console.WriteLine("Ви ще дитина.");
}
else if (age < 18)
{
    Console.WriteLine("Ви підліток.");
}
else
{
    Console.WriteLine("Ви доросла людина.");
}

В цьому прикладі, якщо вік менше 12, виводиться "Ви ще дитина." Якщо вік більше або рівний 12 і менше 18, виводиться "Ви підліток." В іншому випадку, виводиться "Ви доросла людина."

Умовний оператор else:

Оператор else використовується, щоб виконати код, якщо жодна з умов в операторах if і else if не виконується. Він є необов'язковим і може бути використаний тільки після if або else if.

Приклад:

int score = 75;

if (score >= 90)
{
    Console.WriteLine("Ви отримали відмінну оцінку.");
}
else if (score >= 70)
{
    Console.WriteLine("Ви отримали задовільну оцінку.");
}
else
{
    Console.WriteLine("Ви не впоралися з матеріалом.");
}

В цьому прикладі, якщо бали менше 70, виводиться "Ви не впоралися з матеріалом."

Цикли

Огляд циклів for, while та foreach.

Цикл for:

Цикл for використовується для повторення певного блоку коду певну кількість разів. Він має спеціальну змінну, яку зазвичай називають лічильником, і він керується трьома важливими параметрами: ініціалізацією, умовою продовження і кроком. Синтаксис виглядає так:

for (ініціалізація; умова; крок)
{
    // Код, який виконується кілька разів
}

Приклад:

for (int i = 1; i <= 5; i++)
{
    Console.WriteLine("Це ітерація номер " + i);
}

У цьому прикладі цикл for виведе повідомлення п'ять разів, змінюючи i з 1 до 5.

Цикл while:

Цикл while використовується для повторення блоку коду, доки певна умова залишається істинною. Синтаксис виглядає так:

while (умова)
{
    // Код, який виконується, доки умова істинна
}

Приклад:

int number = 1;

while (number <= 5)
{
    Console.WriteLine("Це ітерація номер " + number);
    number++;
}

Цей приклад виведе повідомлення п'ять разів, доки number залишається менше або рівним 5.

Цикл foreach:

Цикл foreach використовується для ітерації по колекціях, таким як масиви або списки, і автоматично перебирає всі елементи цієї колекції. Синтаксис виглядає так:

foreach (тип_елементу змінна_елементу in колекція)
{
    // Код, який виконується для кожного елемента колекції
}

Приклад:

string[] names = { "Анна", "Василь", "Дмитро", "Ірина" };

foreach (string ім'я in names)
{
    Console.WriteLine("Привіт, " + ім'я);
}

У цьому прикладі цикл foreach перебирає всі імена в масиві names та виводить привітання для кожного з них.
