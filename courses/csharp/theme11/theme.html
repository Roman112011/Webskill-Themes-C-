Функції та класи - повторення

Функції (Functions)

Функція - це блок коду, який виконує певну дію або обчислення при виклику.

Функції (Functions):Функція - це блок коду, який виконує певну дію або обчислення при виклику. Функції допомагають уникнути дублювання коду і організують програму в більш логічний спосіб.

using System;

class Program
{
    static void Main()
    {
        // Функція, яка додає два числа та повертає результат
        int sum = Add(5, 3);
        Console.WriteLine("Сума: " + sum);
    }

    static int Add(int a, int b)
    {
        return a + b;
    }
}

У цьому прикладі ми створили функцію Add, яка додає два числа та повертає результат. Ми викликали цю функцію з Main і вивели результат на екран.

Класи (Classes)

Класи - це шаблони для створення об'єктів. Вони описують структуру та поведінку об'єктів.

Класи (Classes):Класи - це шаблони для створення об'єктів. Вони описують структуру та поведінку об'єктів.

using System;

// Приклад класу "Собака"
class Dog
{
    public string Name;
    public int Age;

    public void Bark()
    {
        Console.WriteLine(Name + " гавкає!");
    }
}

class Program
{
    static void Main()
    {
        // Створення об'єкта класу "Собака"
        Dog myDog = new Dog();
        myDog.Name = "Рекс";
        myDog.Age = 3;

        Console.WriteLine("Ім'я собаки: " + myDog.Name);
        Console.WriteLine("Вік собаки: " + myDog.Age);

        myDog.Bark(); // Виклик методу "Bark" класу "Собака"
    }
}

У цьому прикладі ми створили клас "Собака", який має поля для імені та віку собаки, а також метод "Bark", який виводить повідомлення про гавкання собаки. Ми створили об'єкт цього класу та використали його методи та властивості.

Класи та функції на практиці

Приклад програми яка використовує класи та функції.

Опис програми:Кожен студент має ім'я, вік та оцінки з різних предметів. Можна додавати нових студентів, виводити інформацію про студентів, та розраховувати середній бал для кожного студента.

using System;
using System.Collections.Generic;

class Student
{
    public string Name { get; set; }
    public int Age { get; set; }
    public List<int> Grades { get; set; }

    public Student(string name, int age)
    {
        Name = name;
        Age = age;
        Grades = new List<int>();
    }

    public double CalculateAverageGrade()
    {
        if (Grades.Count == 0)
        {
            return 0.0;
        }
        int sum = 0;
        foreach (int grade in Grades)
        {
            sum += grade;
        }
        return (double)sum / Grades.Count;
    }
}

class Program
{
    static void Main()
    {
        List<Student> students = new List<Student>();

        // Додавання нових студентів
        students.Add(new Student("Марія", 15));
        students.Add(new Student("Петро", 14));
        students.Add(new Student("Оксана", 16));

        // Додавання оцінок
        students[0].Grades.Add(95);
        students[0].Grades.Add(88);
        students[1].Grades.Add(76);
        students[1].Grades.Add(85);
        students[2].Grades.Add(90);
        students[2].Grades.Add(92);

        // Виведення інформації про студентів та їх середні бали
        foreach (Student student in students)
        {
            Console.WriteLine("Ім'я: " + student.Name);
            Console.WriteLine("Вік: " + student.Age);
            Console.WriteLine("Середній бал: " + student.CalculateAverageGrade());
            Console.WriteLine();
        }
    }
}

Пояснення роботи програми:

  1. Створюємо клас Student, який має властивості для імені, віку та списку оцінок.

  2. В класі Student визначаємо конструктор, який приймає ім'я та вік студента. Створюємо також список оцінок.

  3. Додаємо метод CalculateAverageGrade, який розраховує середній бал для студента на основі введених оцінок.

  4. У Main, створюємо список студентів students.

  5. Додаємо трьох студентів і додаємо для кожного з них оцінки.

  6. Виконуємо ітерацію по списку студентів, виводячи інформацію про кожного студента та їх середні бали.
